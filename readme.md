# 介绍
基于nodejs平台来完成一个基础功能包括：服务端、客户端的全栈站点
- 后端使用express+mongodb
- 前端使用react

# 技术栈
mongodb+react+express+nodejs

# 安装
> yarn start 启动api和src服务，支持热加载

> yarn build 生成dist文件夹
